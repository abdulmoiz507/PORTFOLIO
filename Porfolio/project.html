Python Game Beginner

Number Guessing Game

A console-based Python game where the computer picks a random number between 1 and 100 and the player tries to guess it. The game gives hints ("too high" / "too low") after each guess and tracks the number of attempts. The player wins when they guess correctly, and the final score is based on how few guesses were needed.

Concepts practised: random module, while loops, if/elif/else conditionals, user input with int() casting, input validation using try/except, and f-string formatting.

How to run: Save the file as number_game.py and run python number_game.py in your terminal. No extra libraries required.

number_game.py 
import random

def get_score(attempts):
    if attempts <= 3:
        return "PERFECT  - Elite Hacker Mode"
    elif attempts <= 6:
        return "GREAT    - Sharp Instincts"
    elif attempts <= 10:
        return "GOOD     - Keep Practising"
    else:
        return "KEEP TRYING - You'll get better!"

def play_game():
    print("=" * 45)
    print("   NUMBER GUESSING GAME")
    print("   Guess the number between 1 and 100")
    print("=" * 45)

    secret = random.randint(1, 100)
    attempts = 0
    guessed = False

    while not guessed:
        try:
            guess = int(input("\nEnter your guess: "))
        except ValueError:
            print("  Please enter a valid number.")
            continue

        attempts += 1

        if guess < 1 or guess > 100:
            print("  Out of range! Guess between 1 and 100.")
        elif guess < secret:
            print(f"  Too LOW!  (Attempt {attempts})")
        elif guess > secret:
            print(f"  Too HIGH! (Attempt {attempts})")
        else:
            guessed = True
            print("\n" + "=" * 45)
            print(f"  CORRECT! The number was {secret}.")
            print(f"  You guessed it in {attempts} attempt(s).")
            print(f"  Score: {get_score(attempts)}")
            print("=" * 45)

    again = input("\nPlay again? (yes/no): ").strip().lower()
    if again == "yes":
        play_game()
    else:
        print("\nThanks for playing. Goodbye!\n")

play_game()
Scratch Game Visual Programming

Interactive Scratch Game

Built using the MIT Scratch editor (scratch.mit.edu), this is a simple interactive game where a character moves across the screen, avoids obstacles, and scores points by collecting items. Scratch uses a visual block-based programming language that teaches the fundamentals of event-driven programming, sprites, and game states without writing traditional code.

Concepts practised: event broadcasting, sprite movement with arrow-key controls, collision detection using "touching color" blocks, variables for score and lives, and "forever" loop blocks for continuous game logic.

How to access: Go to scratch.mit.edu, sign in, and open the project from your profile. You can also recreate it by following the block logic described below.

scratch_game_logic.txt (pseudocode — replicate as Scratch blocks) 
=== SPRITE: Player (Cat) ===

WHEN [Green Flag] clicked:
    set x to -180
    set y to 0
    set [Score] to 0
    set [Lives] to 3

WHEN [right arrow] key pressed:
    change x by 10

WHEN [left arrow] key pressed:
    change x by -10

WHEN [up arrow] key pressed:
    change y by 10

WHEN [down arrow] key pressed:
    change y by -10

=== SPRITE: Coin (Yellow circle) ===

WHEN [Green Flag] clicked:
    forever:
        go to random position
        wait 2 seconds

WHEN touching [Player]:
    change [Score] by 1
    play sound [pop]
    go to random position

=== SPRITE: Obstacle (Red block) ===

WHEN [Green Flag] clicked:
    forever:
        glide 1 secs to random position
        if touching [Player] then:
            change [Lives] by -1
            say [Ouch!] for 1 second
            if [Lives] = 0 then:
                broadcast [Game Over]

=== BACKDROP: Game Over screen ===

Airline Seat Reservation System

A console-based C++ application that simulates an airline seat reservation system. The plane has 30 seats arranged in rows of 3 (First Class: rows 1–5, Economy: rows 6–10). Users can view the seat map, book a seat by entering their name and seat number, cancel a booking, and view all current reservations. Seats are stored in an array of structs, each tracking the seat number, passenger name, and booking status.

Concepts practised: structs, arrays, for-loops, switch statements, string handling with std::string, input validation, and menu-driven program design.

How to run: Save as airline.cpp, compile with g++ airline.cpp -o airline, then run ./airline (Linux/Mac) or airline.exe (Windows).

airline.cpp 
#include <iostream>
#include <string>
#include <iomanip>
using namespace std;

const int TOTAL_SEATS = 30;

struct Seat {
    int    seatNumber;
    string passengerName;
    bool   isBooked;
};

// Initialize all seats as empty
void initSeats(Seat seats[]) {
    for (int i = 0; i < TOTAL_SEATS; i++) {
        seats[i].seatNumber    = i + 1;
        seats[i].passengerName = "";
        seats[i].isBooked      = false;
    }
}

// Display the seat map
void displaySeats(Seat seats[]) {
    cout << "\n============================================\n";
    cout << "        AIRLINE SEAT MAP\n";
    cout << "        [B] = Booked  [A] = Available\n";
    cout << "============================================\n";
    cout << "  FIRST CLASS (Rows 1-5)\n";
    for (int i = 0; i < 15; i++) {
        if (i % 3 == 0) cout << "  Row " << setw(2) << (i / 3 + 1) << ":  ";
        cout << setw(3) << seats[i].seatNumber
             << (seats[i].isBooked ? "[B]" : "[A]") << "  ";
        if ((i + 1) % 3 == 0) cout << "\n";
    }
    cout << "\n  ECONOMY CLASS (Rows 6-10)\n";
    for (int i = 15; i < TOTAL_SEATS; i++) {
        if (i % 3 == 0) cout << "  Row " << setw(2) << (i / 3 + 1) << ":  ";
        cout << setw(3) << seats[i].seatNumber
             << (seats[i].isBooked ? "[B]" : "[A]") << "  ";
        if ((i + 1) % 3 == 0) cout << "\n";
    }
    cout << "============================================\n";
}

// Book a seat
void bookSeat(Seat seats[]) {
    int seatNum;
    cout << "\nEnter seat number to book (1-" << TOTAL_SEATS << "): ";
    cin  >> seatNum;

    if (seatNum < 1 || seatNum > TOTAL_SEATS) {
        cout << "Invalid seat number.\n";
        return;
    }

    Seat &s = seats[seatNum - 1];
    if (s.isBooked) {
        cout << "Seat " << seatNum << " is already booked by "
             << s.passengerName << ".\n";
        return;
    }

    cout << "Enter passenger name: ";
    cin.ignore();
    getline(cin, s.passengerName);
    s.isBooked = true;

    string cls = (seatNum <= 15) ? "First Class" : "Economy";
    cout << "\nBooking confirmed!\n";
    cout << "  Passenger : " << s.passengerName << "\n";
    cout << "  Seat No.  : " << seatNum << "\n";
    cout << "  Class     : " << cls << "\n";
}

// Cancel a booking
void cancelSeat(Seat seats[]) {
    int seatNum;
    cout << "\nEnter seat number to cancel (1-" << TOTAL_SEATS << "): ";
    cin  >> seatNum;

    if (seatNum < 1 || seatNum > TOTAL_SEATS) {
        cout << "Invalid seat number.\n";
        return;
    }

    Seat &s = seats[seatNum - 1];
    if (!s.isBooked) {
        cout << "Seat " << seatNum << " is not booked.\n";
        return;
    }

    cout << "Cancelling booking for " << s.passengerName
         << " on seat " << seatNum << "...\n";
    s.passengerName = "";
    s.isBooked      = false;
    cout << "Booking cancelled successfully.\n";
}

// View all reservations
void viewReservations(Seat seats[]) {
    cout << "\n============================================\n";
    cout << "        CURRENT RESERVATIONS\n";
    cout << "============================================\n";
    cout << left << setw(8) << "Seat"
              << setw(10) << "Class"
              << "Passenger\n";
    cout << "--------------------------------------------\n";

    bool any = false;
    for (int i = 0; i < TOTAL_SEATS; i++) {
        if (seats[i].isBooked) {
            string cls = (i < 15) ? "First" : "Economy";
            cout << left << setw(8) << seats[i].seatNumber
                       << setw(10) << cls
                       << seats[i].passengerName << "\n";
            any = true;
        }
    }
    if (!any) cout << "No reservations found.\n";
    cout << "============================================\n";
}

int main() {
    Seat seats[TOTAL_SEATS];
    initSeats(seats);

    cout << "\n  Welcome to Abdul Moiz Airlines\n";

    int choice;
    do {
        cout << "\n============================================\n";
        cout << "  MAIN MENU\n";
        cout << "  1. View Seat Map\n";
        cout << "  2. Book a Seat\n";
        cout << "  3. Cancel a Booking\n";
        cout << "  4. View All Reservations\n";
        cout << "  5. Exit\n";
        cout << "============================================\n";
        cout << "  Enter choice: ";
        cin  >> choice;

        switch (choice) {
            case 1: displaySeats(seats);    break;
            case 2: bookSeat(seats);        break;
            case 3: cancelSeat(seats);      break;
            case 4: viewReservations(seats); break;
            case 5: cout << "\nThank you. Safe travels!\n"; break;
            default: cout << "Invalid option. Try again.\n";
        }
    } while (choice != 5);

    return 0;
}
